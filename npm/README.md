## 🚀 Installation

### NPM (Universal - Recommended)

```bash
npm install -g siby-agentiq
```

### Build from Source

```bash
git clone https://github.com/siby-agentiq/siby-agentiq.git
cd siby-agentiq
make install
```

### 🦂 SCORPION - Deep Web Search
Multi-API search engine that queries Gemini, GPT-4o, and Perplexity simultaneously.
- Yellow/black progress bar
- Source comparison
- Confidence scoring

### 🧬 EVOLUTION-CORE - Self-Learning Engine
Recursive learning system that:
- Stores every bug fix and lesson learned
- Analyzes performance nightly
- Optimizes prompts automatically
- Builds knowledge base over time

### 👁️ GOD-IA - Omniscient System Vision
Secret mode activated with `leader-siby`:
- Real-time CPU/Memory monitoring
- Process management
- Network connection tracking
- Disk usage analysis
- Code validation for 2035 standards

### 🌈 HOLOGRAM - Visual Mode
ASCII art visualization with multiple themes:
- Cyberpunk
- Terminal
- Neon
- Animated status displays

### 🎤 VOICE - Voice Commands (Coming Soon)

- Synchronize memory across devices
- AES-256-GCM encryption
- Secure multi-device support

## 📦 NPM Distribution

The NPM package automatically:
1. Downloads the correct binary for your OS
2. Sets up configuration in `~/.siby/`
3. Registers the `siby` command globally

### Binary Support
- `siby-agentiq-linux-x64`
- `siby-agentiq-linux-arm64`
- `siby-agentiq-darwin-x64`
- `siby-agentiq-darwin-arm64`
- `siby-agentiq-win32-x64.exe`
